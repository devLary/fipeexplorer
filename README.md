# fipeexplorer 🚗

This is a practical project developed in Java with **Spring Boot**, created with the main purpose of consolidating knowledge in consuming REST APIs and handling JSON data.

## 🎯 Learning & Practice Objectives
* **Spring Boot:** Creating and structuring an application utilizing the Spring ecosystem.
* **HTTP API Consumption:** Implementing requests to fetch real-time vehicle data from the FIPE API.
* **JSON Manipulation with Jackson:** Utilizing the Jackson library to map and deserialize API responses directly into Java objects (*Records*).
* **Semantic Versioning:** Applying Git best practices using the *Conventional Commits* standard to keep the history clear and professional.

## 🛠️ Technologies & Tools
* **Java**
* **Spring Boot**
* **Jackson** (JSON data mapping)
* **Git**
